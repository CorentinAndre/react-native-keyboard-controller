# react-native-keyboard-events

Platform agnostic keyboard manager

## Installation

```sh
npm install react-native-keyboard-events
```

## Usage

```js
import { KeyboardEventsView } from "react-native-keyboard-events";

// ...

<KeyboardEventsView color="tomato" />
```
